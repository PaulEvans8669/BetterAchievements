Better Achievements
▬▬▬▬▬▬▬

Description

▬▬▬▬▬▬▬

Better Achievements is a simple plugin that makes Achievements more intresting for players. The plugin gives to player a Speed and Jump bonus for 2minutes. The player will hear a sound, a message will also be displayed.
